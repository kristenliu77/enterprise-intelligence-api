# 商势智引｜商业热度决策平台

Business Heat Intelligence Platform 是面向地方政府、产业园区、投资促进机构和招商团队的商业热度计算与招商决策辅助前端 MVP。

当前版本使用高质量模拟数据，不连接真实政务、工商、百度指数或第三方付费接口。所有演示结论仅用于产品演示，不能替代人工尽职调查。

## 页面结构

- `/login`：登录页，含表单验证、错误提示、登录加载和路由保护。
- `/dashboard`：招商驾驶舱，含筛选、核心指标、区域热力、排行榜、趋势、机会推荐和机会矩阵。
- `/heat-calculation`：热度计算中心，含权重方案、自定义权重校验、真实前端计算、预测和批量对比。
- `/industry-region`：产业与区域分析，含产业链图谱和区域竞争力对比。
- `/entities`：企业项目库，含搜索、筛选、排序、分页、详情、收藏和 CSV 导出。
- `/matching`：智能匹配，支持区域匹配企业、企业匹配区域。
- `/policies-reports`：政策库与报告中心，支持政策筛选、详情、CSV 导出和浏览器打印 PDF。
- `/workbench`：招商工作台，含线索看板、新建线索和阶段流转。
- `/data-management`：数据管理，含数据源状态、模拟同步、CSV 上传和数据预览。
- `/settings`：系统设置，展示产品配置、角色示例和合规说明。

## 热度计算模型

热度计算位于 `src/services/heatEngine.ts`，包含：

- 正向指标归一化与反向归一化；
- Z-score 扩展函数；
- 指数时间衰减；
- 搜索关注、媒体传播、投资信号、政策匹配、产业基础、综合风险六大维度；
- 综合热度、风险因子、质量因子、趋势因子；
- 数据可信度与热度等级；
- 综合招商、产业趋势、项目落地三套权重方案；
- 自定义权重合计 100% 校验。

## 模拟数据

模拟数据位于 `src/mock/data.ts`，包含：

- 8 个重点产业；
- 12 个区域；
- 30 家企业；
- 20 个招商项目；
- 15 条招商政策；
- 12 个月热度趋势；
- 30 条招商线索；
- 多组搜索、媒体、投资、政策、风险和数据质量指标。

数据为稳定预置数据，避免刷新后随机变化。页面均明确标注“演示数据”。

## 后续接入真实 API

建议保留当前前端接口边界：

- `src/services/mockApi.ts` 替换为真实 API 客户端；
- `src/services/heatEngine.ts` 保留纯计算能力，由后端或前端共同复用；
- 数据源适配器按工商、投融资、搜索趋势、新闻媒体、政策文件、园区内部数据分别拆分；
- 所有接口返回值按 `src/types/index.ts` 中的类型契约对齐。

## 可扩展后端架构

后续可采用：

- FastAPI 作为 API 网关和计算服务；
- Redis 缓存外部接口结果；
- Celery/APScheduler 处理周期刷新；
- PostgreSQL 存储企业、项目、政策、报告和操作日志；
- 对接统一身份认证和角色权限控制；
- 对导出、查看敏感字段等行为做审计记录。

## 数据安全注意事项

- 当前系统为招商决策辅助工具；
- 计算结果不能替代人工尽职调查；
- 演示数据不代表真实政府统计结果；
- 敏感数据需要脱敏处理；
- 用户只能访问其权限范围内的数据；
- 数据导出行为应记录日志；
- 政策变化可能影响分析结论。
